# OhMeter Support

If you need help with OhMeter, please use one of these support channels:

- GitHub Issues: <https://github.com/sapplex-sz/ohmeter/issues>
- Email: <sapplex@qq.com>

When reporting a problem, please include:

- OhMeter version
- macOS version
- Whether the app is installed from the Mac App Store or a local build
- A short description of what you clicked and what happened

OhMeter reads local Codex usage records only after you authorize the folder that contains Codex data.
